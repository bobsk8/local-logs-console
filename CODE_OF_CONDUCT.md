# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus on collaboration and technical quality.
- Accept feedback professionally.

## Unacceptable Behavior

- Harassment, discrimination, or abusive language.
- Personal attacks or intimidation.
- Publishing private information without permission.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Scope

This code of conduct applies to project spaces, issue trackers, pull requests, and related community discussions.
